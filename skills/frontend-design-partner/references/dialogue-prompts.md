# Dialogue Prompts Reference

## Initial Questions

- 主に誰が使うUIか。
- その人が最初の60秒で完了したい操作は何か。
- 絶対に避けたい印象は何か（例: 固い、子供っぽい、情報過多）。
- どこまで壊してよいか（軽微 / 中規模 / 全面）。

## Direction Proposal Format

各案で以下を埋める。

- コンセプト名:
- キーワード（3語）:
- 体験ゴール:
- 主要スタイル（色、タイポ、密度、モーション）:
- 実装コスト（S/M/L）:
- リスク:

## Iteration Questions

- この案で「好きな点」を1つ。
- この案で「違和感がある点」を1つ。
- 次ラウンドで優先的に調整したいのは何か。
